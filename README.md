Sales Performance Analysis Project

Project Overview

This project involved analyzing a transactional sales dataset to identify key performance trends
across different products and sales representatives. The analysis focuses on calculating total
revenue and average transaction values using Excel-based data manipulation and visualization
techniques.

Key Skills Demonstrated

● Data Summarization (Pivot Tables & Cross-Tab Reports)
● Statistical Calculation (Total & Average Transaction Values)
● Data Visualization (Horizontal Bar Charts with custom branding)
● Excel Formatting & Data Integrity

Analysis Results

The analysis yielded two primary reports:
Report Type Primary Insight
Total Sales Identified top-performing sales representatives

by product category.

Average Transactional Sales Calculated the mean value per sale to
understand individual representative efficiency.
Detailed visualizations for these reports were created to provide a clear, professional summary
of the data, as seen in the project files.
